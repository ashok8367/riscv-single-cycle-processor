# RISC-V Single Cycle Processor (RV32I)

This project implements a 32-bit RISC-V single cycle processor using Verilog.

## Features
- RV32I base instruction set
- Instruction fetch, decode, execute, memory, writeback
- Assembly firmware verification
- Simulation using Icarus Verilog and GTKWave

## Supported Instructions
- add
- sub
- addi
- lw
- sw
- beq
- jal
